<h3>PowerShell Script: Bulk User Creation</h3>

<p>
To simulate a real organizational environment, multiple user accounts are created in <b>Active Directory</b> using a <b>PowerShell automation script</b>. This script automatically generates user accounts and places them into the <b>_EMPLOYEES</b> Organizational Unit.
</p>

<p>
The PowerShell script used in this lab can be found in a separate repository:
</p>

<p>
<a href="https://github.com/chrisdegutis/ad-user-creation-script/blob/main/create-active-directory-employees.ps1">View the Bulk User Creation Script</a>
</p>

<p>
Using PowerShell to automate user creation demonstrates how administrators manage large numbers of accounts efficiently in enterprise environments instead of creating users manually.
</p>
